# Retail Sales Analysis Dashboard

## Project Overview

The Retail Sales Analysis Dashboard is a data analytics project developed to analyze retail sales data and generate meaningful business insights. The project combines Python-based data processing with an interactive dashboard to visualize sales performance, revenue trends, category-wise analysis, and regional performance.

## Problem Statement

Retail businesses generate large volumes of sales data, making it difficult to manually track performance and identify business opportunities. This project provides an automated solution to analyze retail sales data and present insights through an interactive dashboard.

## Objectives

- Analyze retail sales transactions.
- Calculate key business KPIs.
- Identify sales trends and patterns.
- Compare performance across product categories and regions.
- Visualize data using interactive charts and dashboards.
- Support data-driven business decision-making.

## Technologies Used

- Python
- HTML5
- CSS3
- JavaScript
- Chart.js
- JSON

## Features

- KPI Dashboard
- Monthly Revenue Analysis
- Category-wise Sales Analysis
- Region-wise Performance Analysis
- Top Products Analysis
- Weekly Sales Trends
- Interactive Data Visualizations

## Key Metrics

- Total Revenue
- Gross Profit
- Profit Margin
- Average Order Value
- Total Orders
- Units Sold
